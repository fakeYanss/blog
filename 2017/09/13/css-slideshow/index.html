css轮播图

本文介绍一种css实现首页轮播图效果的方法，主要用css3的animation属性。

本文转自知乎

由于css无法做到js一样的精准操控，所有某些效果是无法实现的，比如在轮播的同时支持用户左右滑动，所以使用css只能实现基本的效果。下面列出来的内容就是我们实现的：

  1. 在固定区域中，内部内容自行滑动切换形成播放的效果
  2. 当切换到最后一张内容时，会反向播放或者回到起点重播
  3. 每张内容会停留一段时间，让用户能够看清楚
  4. 内容可以点击/进行操作
